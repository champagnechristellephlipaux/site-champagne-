# Checklist post-livraison — Expérience immersive

## 1) Tests UX
- [ ] Mobile : ouverture/fermeture du menu (bouton hamburger)
- [ ] Menu : liens fonctionnent, fermeture au clic, fermeture à la touche Échap
- [ ] Animations : apparaissent progressivement (ou désactivées si préférence réduit mouvement)
- [ ] Scroll : fluide, aucune section écrasée

## 2) Tests boutique
- [ ] Ajout au panier + ouverture panier
- [ ] Paiement Stripe : bouton/redirect OK
- [ ] Retour boutique après paiement (si applicable)
- [ ] Page merci : texte suivi présent

## 3) Performance & accessibilité
- [ ] Vérifier sur mobile 4G : images hero chargent correctement
- [ ] Vérifier focus clavier : outlines visibles sur liens/boutons
- [ ] Vérifier contraste : lisible sur images (hero)

## 4) SEO / indexation
- [ ] `robots.txt` accessible
- [ ] `sitemap.xml` accessible
- [ ] Vérifier 1 seul H1 par page (source HTML)
- [ ] (Après déploiement) soumettre sitemap à Google Search Console
